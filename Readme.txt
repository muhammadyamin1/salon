
## 🧑‍💼 Penggunaan

1. Clone repo ini
2. Upload ke hosting atau jalankan di localhost
3. Pastikan folder `forms/` memiliki izin tulis jika digunakan untuk email
4. Edit bagian kontak dan sosial media sesuai kebutuhan salon
